▶ Mode offline akan dihapus.

Apakah kita harus mengkhawatirkan tentang itu sekarang?

 Walaupun offline mode belum dipermintakan untuk dihapus oleh Mojang, itu masih sama saja piracy waktu anda menggunakannya, karena anda memainkan permainannya secara gratis (kecuali anda membelinya). Karena GitHub dan Mojang dipunya oleh perusahaan yang sama, sekarang waktunya untuk mengumumkan bahwa kita menurunkan kaki kita.

Di masq depan, offline mode akan diganti oleh sebuah mode baru yang dinamai local mode. Akun mode local mempunyai restriksi sama dengan mode offline, ditambah beberapa restriksi baru:

• Akun mode local akan tetap menjadi akun mode offline, artinya anda masih dilimitasu dengan cara akun mode offline bekerja.

• Akun mode local tudaj akan bisa mendiwnload Minecraft melalui launcher.
