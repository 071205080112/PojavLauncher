▶ 1.18: Sudah Datang

Sejak 21w37a, Mojang menaikkan versi resource pack ke versi 8, yang berakibat patch assets-v0 tidak bisa dipakai. Ini adalah patch untuk versi-versi itu. Cukup mengganti ke assets-v0 sekarang, dan anda siap untuk bermain!
Catatan kecil: Ketika popup tentang 1.17 keluar, pencet tetap main. Mempencet OK akan mengoverwrite patchnya. Catatan kecil tambahan: 1.17.1 atau lebih dulu tidak akan bisa menggunakan patch ini.

Ini caranya untuk membuatnya bekerja. Unduh iniberkas JRE17 sesuai dengan arsitektur CPU anda. arm untuk 32bit, arm64 untuk 64bit, and x86_64 untuk Chromebooks/WSA.
Terserah platform, anda akan butuh untuk mengundug resource pack ini.

▶ Mode offline akan berakhir.

Apa kita harus mengkhawatirkan tentang itu sekarang?

 Walaupun mode offline belum diturunkan dari Mojang, itu masih secara teknis adalah pembajakan waktu anda menggunakannya, karena anda mendapatkan gamenya secara gratis (kecuali anda membelinya). Karena GitHub dan Mojang dipunya oleh perusahaan yang sama, inilah waktunya kita akan menurunkan kaki kita.

Pada saat masa depan, mode offline akan diganti oleh sebuah mode baru yang kita panggil mode lokal. Akun user mode lokal masih mempunyai batasan yang sama dengan akun offline, ditambah beberapa yang baru:

• Akun mode lokal masih akan menjadi akun mode offline, berarti anda masih terbatas dengan batasan cara akun mode offline bekerja.

• Akun mode lokal tidak akan dapat mengunduh Minecraft dari launcher.
